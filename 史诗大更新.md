左边侧栏（sidebar）从上到下分别是Home，Connections，search,和Discover
这次的最大更新就是设计哲学：把每一个功能都设计为一个个小组件，然后在sidebar中
点击某一个功能，就会在页面上出现对应的组件（比如现在就是connection的组件，并且会有状态提示
：当按下某个组件时，侧栏里的对应logo就会填充为黑色），而左下角的“更多”选线点击就会弹出一个
小菜单：从上到小分别是创建新行程，查看行程和connact US，与别的不同，这三个页面点击之后会有单独的页面，点击左上角的woconlogo后就会返回homepage（settings页面也一样）
现在显示的是用户点击了一个tenerife旁的行程，右上角就弹出了行程的详细信息（然后用户可以在左上角的搜索框里搜索目的地，比如toyko，再按一下sidebar里的搜索按键，搜索栏就会消失）

新主页样式：🔲 整体布局
页面分为三部分：
左侧固定侧边栏（Sidebar）：宽度 80px，垂直排列图标。
顶部全局搜索栏（SearchBar）：默认隐藏，位于主内容区顶部，宽度 100%。
中央主内容区（MainContent）：占据剩余空间，包含地图背景，并可叠加显示功能组件或弹窗。
🧭 左侧 Sidebar（从上到下）
每个图标为按钮，点击后激活对应功能。当前激活项图标填充为黑色，其余为灰色轮廓。
表格
ID 图标语义 行为
home 首页 显示 Home 组件（空白或欢迎信息）
connections 连接 在主内容区左侧显示 <ConnectionPanel /> 组件
search 搜索 切换顶部 <SearchBar /> 的显示/隐藏
discover 发现 显示 <DiscoverPanel />（暂未实现，可留空）
✅ 注意：home、connections、discover 是“组件模式”，只切换主内容区内的面板；
而 search 是“UI 控制模式”，控制独立搜索栏的可见性。
⋮ 左下角 “更多” 菜单（More Menu）
位置：Sidebar 底部
点击 ... 图标，弹出浮动菜单（绝对定位），选项如下：
Create New Trip → 跳转到 /create-trip（新页面）
View Trips → 跳转到 /trips（新页面）
Contact Us → 跳转到 /contact（新页面）
🔁 所有独立页面均包含左上角 Wocon Logo，点击返回 /（首页）。
🔍 搜索功能
搜索栏位于主内容区顶部（z-index 高于地图）。
输入框 placeholder: "Search destination..."
用户输入（如 "tokyo"）后，地图应高亮匹配位置（前端可模拟）。
再次点击 sidebar 的 Search 图标 → 搜索栏淡出并隐藏。
